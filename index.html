Bootstrap Coda Clips

... for Panic's Coda users who deal with Bootstrap 3's HTML.

Hello.

I'm a Panic's Coda user, and I work a lot with Bootstrap CSS Framework. "Clips" is one of Coda's features I like. And, I think there should be an easier -- and better -- to work with Bootstrap. Bootstrap Coda Clips is basically a collection of HTML clips based on Bootstrap's Documentation.

All clips are made using Coda 2.5 and Bootstrap v3.2.0*.

Download .zip Download .tar.gz View on GitHub

How to install

  • You need to have Coda install, of course.
  • Download the package in .zip or .tar.gz format.
  • You will have multiple files.
  • To install, you can easily double-click on the .clips file. The clips will be imported to Coda automatically. To confirm, navigate to your Clips.
